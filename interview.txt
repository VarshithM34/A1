decorator - it a design pattern that allows us to modify , wrap or extend the 
            dehaivor of the function or method without perminent changeing its 
            sorce code.
A decorator is simply a function that takes another function as an argument, 
adds some behavior inside a nested "wrapper" function, and returns that wrapper.

seek() is a method used to change the position of the file pointer (or cursor) 
       to a specific place within a file.
       0 - default , beginging 
       1 = current position
       2 = end position


tell() is a method that returns the current position of the file pointer 
        (or cursor) within a file.
